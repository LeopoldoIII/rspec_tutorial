# RSpec example
## Environment setup 
 
 Generate a gem file 

    bundle init

Into your gem file add the following line 

    gem 'rspec'


Then run this line from the terminal

    bundle binstubs rspec-core

To generate Rspec structure run this 

    bin/rspec --init

Finally to run test 

    bin/rspec
